# Portfólio 2026 - Posicionamento Técnico

## 1. Nicho de Atuação

Foco no setor de **B2B SaaS**, desenvolvendo aplicações robustas, escaláveis e com excelente experiência de usuário para resolução de problemas corporativos complexos.

## 2. Stack Tecnológica 2026

- **Linguagem Core:** TypeScript
- **Meta-framework:** Next.js (App Router)
- **Integração de IA:** Vercel AI SDK e OpenAI API
- **Estilização e UI:** Tailwind CSS, Shadcn UI
- **Backend/Banco de Dados:** Node.js, PostgreSQL (Prisma/Drizzle)

## 3. Proposta do Projeto Âncora

**Painel Analítico de Retenção (Churn Predictor):** Uma aplicação B2B SaaS focada em Customer Success que consolida dados de uso de clientes, utilizando o Vercel AI SDK para identificar padrões de risco de cancelamento (churn) e gerar automaticamente planos de ação personalizados para retenção. A ferramenta foge do padrão "CRUD genérico", atacando um problema real de negócios através de análise preditiva embasada por IA.
